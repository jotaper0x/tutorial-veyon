Tutorial Veyon: Uso

Guia completo para monitoramento e gerenciamento de computadores em ambientes educacionais ou corporativos

Este tutorial aborda o uso do Veyon para monitoramento e gerenciamento de computadores em ambientes educacionais ou corporativos. Focaremos nas funcionalidades principais da interface e no gerenciamento de salas.

1 Interface do Veyon

Com o Veyon Master devidamente configurado, o próximo passo é conhecer a sua interface. Para isso, execute o programa clicando no ícone disponível na área de trabalho. Em seguida, a tela principal do Veyon será exibida, conforme ilustrado na figura abaixo.

A interface do Veyon Master é projetada para ser simples de usar, tanto no Windows quanto no Linux. O Veyon possui um menu superior com as principais funcionalidades que permitem o monitoramento e gerenciamento dos computadores da sala de aula.

1.1 Menu superior do Veyon

Monitoramento: Modo padrão que permite o monitoramento de todos os computadores de cada sala de aula. O Veyon permite monitorar e gerenciar várias salas ao mesmo tempo.
Exibir em tela cheia: Exibe a tela da máquina do professor em todos os computadores da sala de aula em modo de tela cheia, desativando os dispositivos de entrada dos alunos.
Exibir em janela: Exibe a tela da máquina do professor em todos os computadores da sala de aula em modo de janela, permitindo que o aluno use o computador para outras tarefas.
Bloquear computador: Bloqueia todos os computadores da sala para chamar a atenção dos alunos. Os dispositivos de entrada são bloqueados e as telas são desligadas.
Exibir tela: Monitora um computador específico na sala de aula, sem interação.
Controlar remotamente: Permite ao professor realizar alterações no computador (por exemplo, instalar um programa) ou ajudar um aluno a realizar uma tarefa.
Ligar: Liga todos os computadores de uma sala remotamente (requer Wake-on-LAN ativado).
Reiniciar: Reinicia todos os computadores de uma sala remotamente.
Desligar: Desliga todos os computadores de uma sala remotamente.
Log in: Realiza login com um usuário específico em todos os computadores de uma sala (útil em salas compartilhadas).
Log off: Realiza logoff em todos os computadores de uma sala.
Mensagem de texto: Envia uma mensagem de texto para todos os computadores de uma sala.
Executar programa: Executa um programa em todos os computadores de uma sala.
Abrir website: Abre um site em todos os computadores de uma sala.
Transferência de arquivo: Envia um arquivo para todos os computadores de uma sala.
Captura de tela: Tira um print da tela de um ou mais computadores.

1.2 Menu inferior do Veyon Master

  • 1 - Lista de salas/computadores: Onde se encontra a lista de salas e computadores adicionados ao Veyon.
  • 2 - Capturas de tela: Onde se encontram as capturas de tela.
  • 3 - Área de busca: Utilizada para buscar computadores por nome ou IP, útil em ambientes com muitos computadores.
  • 4 - Mostrar somente os computadores ligados: Filtra a exibição para mostrar apenas os computadores que estão ligados.
  • 5 - Barra de Zoom: Usada para ampliar ou diminuir o tamanho das miniaturas dos computadores.
  • 6 - Customizar o arranjo: Permite customizar o arranjo das miniaturas dos computadores.
